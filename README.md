# Transfer_gift_perfect_world

Это Python-скрипт для активации подарков и промокодов на платформе [pwonline.ru](https://pwonline.ru).

## Описание

Программа предназначена для автоматизации следующих задач:

1. Получение списка доступных подарков (сундуков).
2. Открытие сундуков с выбором подарков и их активация.
3. Активация промокодов для аккаунта игры.
4. Поддерживает работу с кукисами для авторизации.

## Возможности

* **Получение списка подарков**: Скрипт может автоматически собирать список подарков, доступных для активации.
* **Выбор и активация подарков**: Скрипт откроет сундуки и активирует до 30 предметов.
* **Активация промокодов**: Можно ввести и активировать промокод.
* **Поддержка авторизации через куки**: Все запросы выполняются с использованием куки-файлов для авторизации пользователя.

## Требования

* Python 3.x
* Установленные библиотеки: `requests`, `beautifulsoup4`

Для установки зависимостей используйте pip:

```bash
pip install requests beautifulsoup4
```

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/yourusername/promo-code-activator.git
```

2. Перейдите в директорию проекта:

```bash
cd promo-code-activator
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Использование

1. **Подготовьте куки** для авторизации:

   * Вам необходимо получить куки из браузера и сохранить их в формате JSON. Для этого можно использовать расширение браузера, например, [EditThisCookie](https://chrome.google.com/webstore/detail/editthiscookie/).

2. **Запустите программу**:

```bash
python main.py
```

3. В программе будет предложено:

   * Открывать сундуки и выбирать подарки.
   * Ввести промокод для активации, если это необходимо.

4. **Результаты**:

   * Программа будет информировать вас о том, были ли успешно активированы подарки или промокоды, а также если промокод был уже применен.

## Структура проекта

* `main.py`: Основной скрипт для активации подарков и промокодов.
* `cookies/`: Директория для хранения cookie-файлов.
* `requirements.txt`: Список зависимостей для установки через pip.
